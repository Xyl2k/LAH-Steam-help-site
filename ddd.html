How to Freeze and un-freeze an object

Have you ever had built something really unique and right when u go to touch something everything falls to pieces? Or ever had a house put together but then have somebody come in and steal your props? To prevent this from happening we use commands to freeze objects in one place. To do this u must:

e_freeze – Use this command while pointing at a prop to freeze it in a specific location and angle. This is useful for making houses and building jails.

e_unfreeze – This command works the same as e_freeze but you can unfreeze an object that you have previously frozen. This is for moving an object that you have frozen to a different spot to fit your liking.

Removing Props


There are 2 ways to remove props. One way is only for admins in the Chaos Crew server. I will explain both for Admins/mods will be reading this 2. Removing a prop is very important because the more props there is in a server, the more likely it is for the server to crash. So to prevent this we “delete” them. To remove props use the following commands:

e_kill – THIS IS FOR ADMINS ONLY
This command will shoot a red beam out of you and remove the prop you were pointing at. This is also good for slaying unwanted visitors or people who break the rules if they are in sight.

e_removespawned – this is available for all players.
When typed in console it will remove any props you have spawned
Or copied that are on the map. If you do not wish to remove all of your props but just a few, ask an admin politely and we would
Gladly be able to help.

How to spawn a prop

Utilisé la commande e_spawnprop suivi de l'entité a spwawn derrière pour faire apparaitre un objet.
Exemple:
e_spawnprop turret_floor frozen
Cela spawn une tourelle et devient gelé par la même occasion le "frozen" et optionnel

English:
Spawnable Props for e_spawnprop
Usage: e_spawnprop (propname)
ex: e_spawnprop turret_floor frozen
